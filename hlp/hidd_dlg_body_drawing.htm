Body Properties - Drawing & Computation

Drawing

  • Line Color: Customize body line color. Click the button to display the color dialog box ad choose color. With the line color are drawn body boundaries in Profile View and Map View.
  • Line Width: Specify line width in logical coordinates. This is going to be changed to device coordinates (pixels). If line with is 0, line width is one pixel.
  • Fill color: Specify fill color. Body polygon is filled with this color.
  • Visible: If checked body is visible. Otherwise is hidden.
  • Filled: If checked polygon is filled with fill color. Otherwise only

Computation

  • Loocked: The body is locked for reshaping. You cannot modify body shape.
  • Active: The body is active in computation. If not checked the body is excluded from computation.
  • Virtual body acts neither in model definition nor in computation.
