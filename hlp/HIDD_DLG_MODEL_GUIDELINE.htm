Guideline Properties Dialog

Set guideline properties.

Type: Not Used.

Display: Active in Profile View
Visible: Show or hide guideline arrow indicators.
Show Text: Show or hide guideline text.

Description: Set description for Profile View.

Pen: Click Pen button to adjust pen properties.

 

Gudeline file format:
      keywords: line [name] describes the begining of a guideline; name parameter is optional but recomended and is string shown in Profile View as a description of guideline.
      After line [name] keyword follows data section. Data section can contain n floating point numbers in on line. First coordinate is x followed by y and z coordinates.  First two are necassary for a valid piont. Third, z coordinate can be omited.

***  file "guideline.dat" *******
line sampleGL
100 100 10
150.1 200.5 8.6
....
