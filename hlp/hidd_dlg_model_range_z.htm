Model Vertical Range

Modifies the model vertical range. Vertical range is used internally by the system, when defining the model and displaying the model.

Maximum z coordinate should be over relief. Used for displaying purposes.

Minimum z coordinate should be the lower boundary of the model definition. It is the depth limit for the model definition workspace. Can be modified any time.

Horizontal scale = d * Vertical scale. Controlls the scaling in Profile View. May not be implemented.
