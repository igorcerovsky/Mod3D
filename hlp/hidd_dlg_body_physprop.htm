Body Properties - Physical Properties

Gravity:

  • Density of the body in kg/m 3. The density can be either absolute ( 2675 kg/m3) or relative ( 0.005 kg/m3) dependig on wheather you use reference density model or not. Using absolute density is recommended.
  • Gradient: Enter density gradient for linearly insreasing density in kg/m4. Z axis is pointing upward!
  • Origin is a point where Density value is defined. Set to compute the linealrly increasing density. The density at given point r is computed after r ( r) = r 0 + r g × r, where r 0 is the density at the origin of coordinate system (this is computed from Gradient and Origin parameters, density is "shifted" to the origin of coordinate system from Origin point), r g is the density gradient.

Magnetics:

  • Set magnetic Susceptibility in SI units. Dimensionless number equal to (1- m r) where m ris the magnetic pemeabity of the body relative to the vacuum
  • Set Renanent Magnetization Intensity magnitude of magnetic induction in nT.
  • Set inclination of remanent manetic induction in degrees.
  • Set declination of remanent manetic induction in degrees.
