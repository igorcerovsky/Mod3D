1D Fit Properties

Fitting Field: Select field used for fitting. MUST be difference field.

Method: Select fitting method. Brent method uses parabolic interpolation and should converge faster. Further reading: i.e. Press et al., Numerical Recipes in C.

Log File: Choose, whether you want to create log file. Click the Browse button, to specify the file name. 

Characteristic:  Specify the characteristic used in fitting algorithm.
RMS - Root Mean Square of the difference grid.
Derivative of the difference field grid.
Local Derivative is the derivative of the difference grid in fitting point. In density fitting ignored.

Bracketing epsilon: Starting guess interval for bracketing the minimum.

Tolerance < 3.0e-8: Set the fitting tolerance. This value should be less then square root of the machine's floating point precision (for double type 15 valid decimal digits => minimum meaningful value is 3.0e-8).

Maximum iterations:  Specify maximum number of iterations for Brent's method.

Auto bracketing: Available only for vertex fitting. When checked system computes "optimal" bracketing epsilon. Specify the percent of value of maximum available fit interval.
