Body Properties: Drawing

  • Line: Set the line properties of the body. See
  • Line Next: Set the line properties for this body contours from next profile.
  • Line Prev: Set the line properties for this body contours from previous profile.
  • Fill: Set the fill properties. Fill color is also used id 3D View to draw the body, either wireframe or filled.
  • Transparent: Set the transparency for 3D View. Floating point alpha value MUST fit the interval <0, 1>. For alpha=0 the body is invisible, for alpha=1 the body is opaque.
  • Filled: Specify if the body is filled in Profile View.
  • Visible in 3D: Specify, if the body should be displayed in 3D View.
