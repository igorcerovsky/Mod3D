Well Properties Dialog

Well Data report view shows the well data and is rad only. First two rows display the channel minimum and maximum values.

Lithology report view shows lithology defined for given well. Left mouse button double click opens the color dialog.

Draw Channel report view shows channels with check boxes to specify, which log channel to draw. Left mouse button double click opens the pen dialog. Specify the pen for log data drawing, though. Only log data channels will be active for drawing (x, y, z, Lithology channels are ingnored).

Drawing to Profile View group specifies how the well is drawn to Profile View

  • Display radius [m]  - if the  shortest distance from well to profile is within display radius distance the well is drawn to Profile View.
  • Log Radius [m]  - log data graphs are drawn from well to Log Radius distance.
  • Well width [pixels] - well width in pixels on screen.
  • Align to well - choose one of the values where to align the well log data. The well log graph is drawn to the left, right or centered relative to the  well axis.

3D Drawing specifies how the well is drawn to 3D View.

  • Solid (check box) - if checked well log graph  is drawn solid, otherwise as wire frame.
  • Log Radius [m] - log data  extent from the  center of the well.
  • Well  width [pixels] - well line width in  rasterized pixels.
  • Ring sampling - number of ring segments for optimized drawing.
  • Pie start [deg] - specify the pie  start in degrees.
  • Pie end [deg]- specify the pie  end in degrees.
  • Log sampling - each n-th is drawn.
