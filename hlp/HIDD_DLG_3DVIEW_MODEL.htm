3D Model

Transparent: The bodies are drawn transparently. Set the transparency for given body in Body Properties.

Solid: The model bodies are drawn solid if checked or as wire frame otherwise.

Vertical facets: Specfy whether the vertical facets are to be drawn.

Outer facets: Not active.

Darken inside body: Specify whether the body shoul be darkened inside for better orientation. The inside facets color is darker by specified factor. ColorInside = FacetColor * Factor. If Factor=1.0 inside facets have the same color as outside facets. Factor = 0.0 means, that the inside facets are black.

 

Cull Facets:

  • Apply facet culling. Set whether back or front facets are culled, respectivelly.
  • Back / Front facets to cull, it Apply check box is chcecked.

Constrain model. Set rectangular block of a model to be viewed, constrained by rows or columns extent.
