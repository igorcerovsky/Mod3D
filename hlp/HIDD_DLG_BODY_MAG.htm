Body Properties: Magnetics

Set the physical properties for the magnetic field computation.

  • Susceptibility: Set the susceptibility of the body in [SI] units - dimensionless number equal to (1- m r) where m r is the magnetic permeabity of the body relative to the vacuum.   Magnetic field is computed using the inducing field specified Model / Inducing field menu.
  • Remanent Magnetization - Intensity: Set the intensity of the remanent magnetization in [nT].
  • Remanent Magnetization - Inclination: Set the inclination of the remanent magnetization in [degrees].
  • Remanent Magnetization - Declination: Set the declination of the remanent magnetization in [degrees].
