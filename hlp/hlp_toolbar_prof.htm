Profile Toolbar.

The Profile Toolbar provides quick mouse access to tools used for work with within Profile View . To hide or display the Model Toolbar, choose Model Toolbar from the View menu. The Model Toolbar acts in Profile View, if this is currently selected. If the toolbar is not active, select the Profile View first by clicking inside the view.
